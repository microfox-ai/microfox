The AI Router package is currently in **beta development**. While the core functionality is stable and ready for use, we're actively working on several exciting features to enhance the developer experience and expand the ecosystem.

## Features in Development

1. **Vue & Nuxt Support**: Native adapters for Vue.js and Nuxt.js frameworks
2. **Additional Fullstack Frameworks**: Expanding support beyond Next.js to other popular frameworks
3. **Non-Stream Direct Response Structure**: Support for traditional request-response patterns alongside streaming
4. **Improved Error Handling**: Robust error handling with detailed error messages and recovery strategies
5. **Observability & Debugging**: Neat flow chart UIs for debugging and monitoring AI router flows
6. **Type Safety Improvements**: Enhanced and standardized TypeScript support throughout the package
7. **Auto Paths Typing**: Intelligent autocomplete for `ctx.next()` call triggers to prevent spelling mistakes
8. **Orchestration Examples**: Comprehensive example templates showing different orchestration patterns
9. **Reusable Agent Components**: Shadcn/ui styled approach for serving reusable agents across repositories, enabling collaborative development
10. **Cross-Repository Agents**: Standardized approach for sharing and reusing AI agents across different projects
11. **Collaborative Development Tools**: Enhanced tooling for team-based AI agent development

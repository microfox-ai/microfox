---
title: 'getChannelConversationInfo'
description: 'Retrieves information about a specific channel.'
---

The `getChannelConversationInfo` method retrieves information about a specific channel.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    const channelInfo = await client.getChannelConversationInfo({ channelId: 'C12345678' });
    console.log(channelInfo);
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="channelId" type="string" required>
        The ID of the channel to get information about.
    </Property>
</Properties>

## Response

This method returns a `conversation` object, which contains information about a channel-like thing in Slack.

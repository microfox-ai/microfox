---
title: 'messageChannels'
description: 'Sends a message to multiple channels.'
---

The `messageChannels` method sends a message to multiple channels.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    await client.messageChannels({
      channelIds: ['C12345678', 'C87654321'],
      text: 'Hello, everyone!',
    });
    console.log('Messages sent.');
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="channelIds" type="string[]" required>
        The IDs of the channels to message.
    </Property>
    <Property name="text" type="string" required>
        The text of the message to send.
    </Property>
    <Property name="username" type="string">
        Optional custom username for the message.
    </Property>
    <Property name="icon_url" type="string">
        Optional URL to an image to use as the icon for the message.
    </Property>
</Properties>

## Response

This method returns an array of `ChatPostMessageResponse` objects.

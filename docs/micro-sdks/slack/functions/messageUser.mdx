---
title: 'messageUser'
description: 'Sends a direct message to a user.'
---

The `messageUser` method sends a direct message to a user.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    await client.messageUser({
      userId: 'U12345678',
      text: 'Hello!',
    });
    console.log('Message sent.');
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="userId" type="string" required>
        The ID of the user to message.
    </Property>
    <Property name="text" type="string" required>
        The text of the message to send.
    </Property>
    <Property name="username" type="string">
        Optional custom username for the message.
    </Property>
    <Property name="icon_url" type="string">
        Optional URL to an image to use as the icon for the message.
    </Property>
</Properties>

## Response

This method returns a `ChatPostMessageResponse` object.

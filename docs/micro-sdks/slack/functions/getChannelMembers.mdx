---
title: 'getChannelMembers'
description: 'Lists all users in a specific channel.'
---

The `getChannelMembers` method lists all users in a specific channel.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    const users = await client.getChannelMembers({ channelId: 'C12345678' });
    console.log(users);
  } catch (error)_ {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="channelId" type="string" required>
        The ID of the channel to get the members of.
    </Property>
</Properties>

## Response

This method returns an array of user IDs.

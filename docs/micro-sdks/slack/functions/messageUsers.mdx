---
title: 'messageUsers'
description: 'Sends a direct message to multiple users with optional templating.'
---

The `messageUsers` method sends a direct message to multiple users with optional templating.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    await client.messageUsers({
      userIds: ['U12345678', 'U87654321'],
      text: 'Hi {mention}! This is a broadcast message.',
    });
    console.log('Messages sent.');
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="userIds" type="string[]" required>
        The IDs of the users to message.
    </Property>
    <Property name="text" type="string" required>
        The text of the message to send. Can include template variables for personalization.
    </Property>
    <Property name="username" type="string">
        Optional custom username for the message.
    </Property>
    <Property name="icon_url" type="string">
        Optional URL to an image to use as the icon for the message.
    </Property>
</Properties>

## Response

This method returns an array of `ChatPostMessageResponse` objects.

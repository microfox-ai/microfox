---
title: 'uploadFile'
description: 'Uploads a file to a channel using an external URL.'
---

The `uploadFile` method uploads a file to a channel using an external URL.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';
import * as fs from 'fs';
import * as path from 'path';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    const filePath = path.join(__dirname, 'example.txt');
    const file = fs.readFileSync(filePath);

    await client.uploadFile({
      filename: 'example.txt',
      file,
      channelId: 'C12345678',
    });
    console.log('File uploaded.');
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="filename" type="string" required>
        The name of the file.
    </Property>
    <Property name="file" type="Buffer" required>
        A Buffer containing the file content.
    </Property>
    <Property name="channelId" type="string">
        The ID of the channel to upload the file to.
    </Property>
    <Property name="alt_text" type="string">
        Description of image for screen-reader.
    </Property>
    <Property name="snippet_type" type="string">
        Syntax type of the snippet being uploaded.
    </Property>
    <Property name="initialComment" type="string">
        The message text introducing the file in specified channels.
    </Property>
    <Property name="title" type="string">
        An optional title for the file.
    </Property>
</Properties>

## Response

This method returns a `FilesCompleteUploadExternalResponse` object.

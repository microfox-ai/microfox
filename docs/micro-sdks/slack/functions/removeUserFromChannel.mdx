---
title: 'removeUserFromChannel'
description: 'Removes a user from a channel.'
---

The `removeUserFromChannel` method removes a user from a channel.

## Usage

```typescript
import { MicrofoxSlackClient } from '@microfox/slack';

const client = new MicrofoxSlackClient(process.env.SLACK_BOT_TOKEN);

(async () => {
  try {
    await client.removeUserFromChannel({
      channelId: 'C12345678',
      userId: 'U12345678',
    });
    console.log('User removed.');
  } catch (error) {
    console.error(error);
  }
})();
```

## Arguments

<Properties>
    <Property name="channelId" type="string" required>
        The ID of the channel to remove the user from.
    </Property>
    <Property name="userId" type="string" required>
        The ID of the user to remove.
    </Property>
</Properties>

## Response

This method returns a `ConversationsKickResponse` object.

---
title: 'sendBulkEmails'
description: 'Sends multiple emails in parallel using the provided parameters.'
---

The `sendBulkEmails` function sends multiple emails to a list of recipients using the AWS SES service. Emails are sent in parallel for improved performance.

## Usage

```typescript
const responses = await ses.sendBulkEmails({
  sender: 'sender@example.com',
  recipients: [
    'recipient1@example.com',
    'recipient2@example.com',
    'recipient3@example.com',
  ],
  subject: 'Hello from SES!',
  bodyText: 'This is a plain text email body.',
});

console.log(responses);
```

## Arguments

<Properties>
    <Property name="params" type="object" required>
        Parameters for sending bulk emails.
        <Properties>
            <Property name="sender" type="string" required>
                Email address of the sender (must be verified in AWS SES).
            </Property>
            <Property name="recipients" type="string[]" required>
                List of recipient email addresses.
            </Property>
            <Property name="displayName" type="string">
                Display name for the sender (e.g., "John Doe" &lt;john@example.com&gt;).
            </Property>
            <Property name="subject" type="string" required>
                Subject line of the email.
            </Property>
            <Property name="bodyText" type="string">
                Plain text version of the email body.
            </Property>
            <Property name="bodyHtml" type="string">
                HTML version of the email body.
            </Property>
        </Properties>
    </Property>
</Properties>

## Response

This function returns a `Promise<SendEmailResponse[]>` which resolves to an array of `SendEmailResponse` objects, one for each recipient.
